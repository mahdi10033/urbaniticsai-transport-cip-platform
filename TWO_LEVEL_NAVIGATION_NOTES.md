# Two-Level Navigation Update v14

This version reorganizes the platform into two main analysis levels.

## Portfolio Intelligence

System-wide / agency-wide analysis:
- Executive Overview
- Priority Map
- Top Priorities
- Portfolio Risk Indicators
- Scenario Planning
- Asset Inventory & Needs
- Mobility Performance
- Strategic Alignment
- Capital & Funding Strategy
- Implementation Schedule
- Investment Prioritization
- Deferred Maintenance
- Data Governance

## Project Intelligence

Selected-project analysis:
- Project Profile
- Decision Explainability
- Score Breakdown
- Sub-Score Logic
- AI Planning Narratives
- Download Project Report

## Why this matters

This separates portfolio-level decision support from project-level justification and explainability, making the platform easier to navigate and more professional for agency users.
